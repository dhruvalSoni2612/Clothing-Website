E-commerce Clothing Website

This project is a clone of the Aajio clothing app, built using React, Redux, HTML, CSS, and JavaScript. 
It aims to replicate the user interface and functionality of the Aajio app, providing users with a seamless shopping experience for clothing and accessories.

Users can browse through a wide range of clothing and accessories categorized by type, brand, and price. 
Add items to your cart, update quantities, and proceed to checkout to complete your purchase. Also add or remove products from Wishlist.
